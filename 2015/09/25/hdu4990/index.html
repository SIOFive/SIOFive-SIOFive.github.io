HDU 4990 Reading comprehension（分奇偶讨论+矩阵快速幂）

2015.09.25 SIO__Five ACM  热度 ℃

题目

源地址：http://acm.hdu.edu.cn/showproblem.php?pid=4990

题意

给定n，按照以下程序算出ans。（n为int）
ans=0;
for(i=1;i<=n;i++)
{
if(i&1)ans=(ans2+1)%m;
else ans=ans2%m;
}
printf(“%d\n”,ans);

思路

分奇偶讨论
当n为奇数时：fn=4fn-1 + 1，首项为1
当n为偶数时：fn=4fn-1 + 2，首项为2
之后就矩阵快速幂加速运算即可。
