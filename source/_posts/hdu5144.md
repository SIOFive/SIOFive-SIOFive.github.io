# 题目	
源地址：http://acm.hdu.edu.cn/showproblem.php?pid=5144

# 题意
给定一个高度H, 从H处抛出一个初速度为v的小球，问小球最远能到哪里。

# 思路
可以直接推公式
vtcos(0) = s
vtsin(0) = 0.5*g*t^2 - h
把角度消去，得到一个关于t^2的二次方程，要是这个方程有解，判别式必须≥0。所以s = v * sqrt(2.0 * g * h + v * v) / g;

也可以直接三分s（比三分角度精确）

```
#include<iostream>

    int t;

    int h, v;

        printf("%.2f\n", s);
    }
    return 0;
}
```
